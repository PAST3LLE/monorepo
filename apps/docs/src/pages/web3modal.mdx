# PstlW3Providers

The `PstlW3Providers` component manages configuration for all hooks using [React Context](https://reactjs.org/docs/context.html).

```ts
import { PstlW3Providers } from '@past3lle/web3-modal'
```

## Usage

```tsx
import { PstlW3Providers } from '@past3lle/web3-modal'
import { CONFIG } from 'my-path-to-config'

function App() {
  return (
    <PstlW3Providers config={CONFIG}>
      <YourRoutes />
    </PstlW3Providers>
  )
}
```

## Configuration

### config

A wagmi [`config`](/react/config) instance that consists of configuration options. Defaults to `createConfig()`.

```tsx {8-12,16}
import { PstlW3Providers, createConfig, configureChains, mainnet } from 'wagmi'
import { publicProvider } from 'wagmi/providers/public'

const { publicClient, webSocketPublicClient } = configureChains(
  [mainnet],
  [publicProvider()],
)

const config = createConfig({
  publicClient,
  webSocketPublicClient,
})

function App() {
  return (
    <PstlW3Providers config={config}>
      <YourRoutes />
    </PstlW3Providers>
  )
}
```
