# **PAST3LLE LABS MONOREPO** 
A collection of React Hooks/Components/Assets/Widgets/Utils/Types containing everything you need to start working with Ethereum.

# Packages

:::tip
## <img className="icon-header" src="/web3.jpeg"/> Web3
:::
Web3-focused packages for React
### [Web3Modal](/web3modal)
Easily wire-up frontends to a fully-customisable, beautifully extensible Web3 modal suite complete with Wagmi/Viem V2 and **transactions tracking** for both regular web3 EOA and Safe multisig transactions.

:::tip
## <img className="icon-header" src="/forge.png"/> FORGE
:::
FORGE focused packages for React. [View more info about FORGE here](https://skills.pastelle.shop)
### [ForgeWeb3](/forgeweb3)
Wire-up any frontend to the FORGE React API complete with [Web3Modal](/web3modal), auto-state updaters, and necessary Wagmi/Viem hooks/utilities.
### [ForgeWidget](/forgewidget)
Easily create configurable FORGE widgets available with full-theming and web3 customistaions
